# Fossils_in_the_Classroom
Storage for digital materials used to build a 2017 project - "Fossils in the Classroom". This allows the K-12 teachers that were distributed the kits to print additional materials or reprint materials.

Within this repository you will find: 
(1) Fossils in the Classroom 2017 - contains all information used to create the 2017 fossil kits (supplies, images, etc.)
(2) Darwin Day 2016 Teachers Workshop - contains all information used to execute DD 2016 workshop with a theme of paleontology (lesson plans, informational handouts, puzzles, fossil kit information, etc.)
