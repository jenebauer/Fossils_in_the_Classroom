# Fossils_in_the_Classroom
Storage for digital materials used to build a 2017 project - "Fossils in the Classroom". This allows the K-12 teachers that were distributed the kits to print additional materials or reprint materials.

Within this repository you will find: 
<ul>
   <li><strong>(1) Fossils in the Classroom 2017</strong> - contains all information used to create the 2017 fossil kits (supplies, images, etc.)</li>
   <li><a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop"><strong>(2) Darwin Day 2016 Teachers Workshop</strong></a> - contains all information used to execute DD 2016 workshop with a theme of paleontology. </li>
   <ul>
      <li>Lesson plans used at the workshop can be found <a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop/DD%202016%20Lesson%20Plans%20Used%20at%20Workshop">here</a></li>
      <li>Additional lesson plans that were prepare for the workshop can be found <a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop/DD%202016%20Additional%20Lesson%20Plans">here</a></li>
      <li>Informational handouts such as why we should teach evolution in classrooms and common misconceptions can be found <a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop/DD%202016%20Educational%20Handouts">here</a> alongside academic papers from open source journals written for the public.</li>
      <li>Fossil kit booklet and images for the interior and exterior of the kit can be found <a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop/DD%202016%20Fossil%20Kit">here</a></li>
      <li>Posters on understanding evolution, fossilization, time and the age of the Earth, paleontology, and stratigraphy can all be found <a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop/DD%202016%20Informational%20Posters">here</a></li>
      <li>Crossword puzzles with a paleontology theme can be found <a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop/DD%202016%20Puzzles">here</a></li>
   </ul>
</ul>
