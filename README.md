# Fossils_in_the_Classroom
Storage for digital materials used to build a 2017 project - "Fossils in the Classroom". This allows the K-12 teachers that were distributed the kits to print additional materials or reprint materials.

Within this repository you will find: 
<ul>
<li><strong>(1) Fossils in the Classroom 2017</strong> - contains all information used to create the 2017 fossil kits (supplies, images, etc.)</li>
<li><a href="https://github.com/jenebauer/Fossils_in_the_Classroom/tree/master/Darwin%20Day%202016%20Teachers%20Workshop"><strong>(2) Darwin Day 2016 Teachers Workshop</strong></a> - contains all information used to execute DD 2016 workshop with a theme of paleontology (lesson plans, informational handouts, puzzles, fossil kit information, etc.)</li>
</ul>
